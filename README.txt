Promo-HomeWork
In this repo you can find 5 simple automated scripts for testing sign in and sign up page for Promo

Prerequisites:
- Node.js installed on your machine

How to run tests:
  1. Save repository on your machine
  2. Open it in your IDE
  3. Run 'npm install' in order to install all dependencies
  4. Run 'npx cypress open' in order to open cypress 
  5. Click on 'E2E Testing' component
  6. Click 'Start E2E Testing in Chrome' button
  
  Now you can open tests by clicking on 'login.cy.js' or 'signUp.cy.js' file
